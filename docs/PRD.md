## 项目概述

**项目名称**: Deep Research Agent

**技术栈**: LangGraph, RAG, FastAPI, Qdrant, LLM

**核心目标**: 构建一个能够自动完成深度研究任务的智能 Agent 系统，支持用户输入自然语言问题，系统输出带有引用来源、结构清晰、逻辑完整的研究报告。

详细需求参见源文件 `参考文档.txt`。
